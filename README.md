# Learn Graphics Design - Landing Page

A responsive landing page for a graphics design course, implemented with modern CSS and JavaScript.

## Features

- **Responsive Design**: Fully responsive layout optimized for Desktop, Tablet, and Mobile devices.
- **Sticky Header**: Navigation header stays visible while scrolling for easy access.
- **Mobile Navigation**: Hamburger menu for mobile devices with smooth slide-in animation.
- **Right Sidebar Drawer**:
  - Accessible via the Grid Icon in the header.
  - Displays a grid of images.
  - Click-outside to close functionality.
- **Modern UI**:
  - Nunito font used.
  - Hover effects on buttons and links.
- **Code Structure**:
  - Refactored CSS into logical modules (`style.css`, `responsive.css`, `sidebar.css`).
  - Semantic HTML5 structure.

## Technologies Used

- **HTML5**: Semantic markup.
- **CSS3**: Flexbox, Grid, CSS Variables, Media Queries, CSS Transitions.
- **JavaScript (ES6)**: DOM manipulation for interactive elements.

## How to Run

1.  Clone the repository or download the files.
2.  Open `index.html` in any modern web browser.
3.  Resize the browser window to test responsiveness.
